### New Chrome Extension

We are excited to introduce the Silicon Psyche ACT Monitor Chrome Extension. This extension aims to enhance user experience by providing monitoring features that will assist users in tracking and understanding their interaction with the ACT core application.

### Key Features
- Easy installation and setup
- Real-time monitoring
- User-friendly interface

For more details, you can check the commit [e29b82716755029e03831a201b1dc8f2a2133df0](https://github.com/SiliconPsycheLabs/ACT-core/commit/e29b82716755029e03831a201b1dc8f2a2133df0) for changes made to support this extension.
